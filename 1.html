HTML이란 무엇인가?

시작 : File을 눌러 새로운 폴더에 web -> 1.html. 새로운 브라우저를 열어 제목을 드래그하여 주소창에 옮긴다. 혹은 ctrl+o.
저장 : ctrl+s

태그(tag) : 설명하는 것. 열리는 태그와 닫히는 태그가 있다. 뒤에는 슬래시를 추가하여 구분한다.
태그 예 : strong(글씨 굵게), u(밑줄), h1~6(제목), br(줄바꿈), p(paragraph)
태그 검색 : HTML h1 tag : 예시를 붙여넣는다.(heading : 제목)
h1~6는 제목을 나타내는 태그이다.(7은 없다.6까지만)

html new line tag : br (끝을 닫을 필요는 없다.)
html paragraph tag : p (끝을 닫아야 한다.) css를 이용하여 문단의 여백을 조절할 수 있다.

속성(attribute) : 태그만으로 부족할 때 사용. 위치는 상관 없음.
img : 이미지 삽입 태그.
src(source)="coding.jpg" width="100%" 라는 attribute를 작성.

ol-ul-li
li : 목차(list)의 태그
ul : unordered list.
ol : ordered list.
*li의 부모태그. li는 ul의 자식태그. 자식태그를 묶을 수 있다.
*표는 table 태그를 이용.

title : 웹페이지의 제목 설정 태그

*charset=character set
